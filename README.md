# Mundo PC
Sistema de gestión de una tienda de PCs desarrollado en Python con POO.

## Clases
- `Computadora` — representa una PC con sus periféricos
- `Monitor`, `Teclado`, `Raton` — componentes de la PC
- `DispositivoEntrada` — clase base para teclado y ratón
- `Orden` — agrupa computadoras en una orden de compra

## Ejecutar
```
python mundo_pc_app.py
```
